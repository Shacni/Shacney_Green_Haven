## Installation

1. Clone the repo:
   ```
   git clone https://github.com/yourusername/shacney-green-haven.git
   cd shacney-green-haven
   ```

2. Run the Flask server:
   ```
   pip install flask
   python server.py
   ```

3. Open in browser at `http://127.0.0.1:5000/`
